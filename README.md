Danbooru desktop client
=================

A program that utilises the Danbooru API, written in Lua using the [AutoPlay Media Studio 8.0][1] IDE.

Supports searching the following imageboards:

  - [Danbooru][2]
  - [Konachan][3]
  - [Moe Imouto][4]
  - [Gelbooru][5]
  - [Sankaku Channel][6]
  - [Nekobooru][7]
